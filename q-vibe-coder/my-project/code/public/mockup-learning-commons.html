üí° Tip Python Basics
Never use mutable default arguments in Python.

This is one of the most common Python gotchas. When you write def foo(items=[]), that empty list is created ONCE when the function is defined, not each time it's called. Every call shares the same list!

Instead, use def foo(items=None): items = items or []
